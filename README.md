# Tribute Page

This project is a **Tribute Page** developed using **HTML and CSS** to honor and showcase information about a person I admire.

The page includes text, images, and a clean layout to present information in an attractive and meaningful way.

This project was created as **Level 2 – Task 2** during my **Oasis Infobyte Web Development Internship**.

---

## 📌 Features

- Simple and clean tribute page design
- Responsive layout
- Organized content sections
- Attractive typography and styling
- Images and descriptive text about the person

---

## 🛠️ Technologies Used

- HTML5
- CSS3
